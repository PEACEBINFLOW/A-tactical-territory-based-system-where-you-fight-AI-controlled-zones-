# AI Territory Strategy (Prototype)

A browser-based prototype for a tactical strategy game where:

- Units fight on a grid battlefield
- Terrain affects gameplay
- Units evolve based on player behavior
- AI enemies respond dynamically

## How to Run

Just open `index.html` in your browser.

## Features

- Grid-based combat
- Enemy AI
- Unit evolution system
- Terrain variation
